# WhatsApp RPG Bot

Bot WhatsApp RPG sederhana untuk Termux dengan Node.js 25.

## Fitur
- Sistem pendaftaran karakter
- Battle melawan monster
- Leveling system
- Inventory system
- Toko untuk membeli item
- Leaderboard
